# krish-portfolio
 PORTFOLIO WEBSITE :
 Designed and developed a personal portfolio website to showcase my skills, projects, and resume.
 The site is built using React JS for component-based structure, Tailwind CSS for responsive and
 utilityfirst styling, and GSAP (GreenSock Animation Platform) to add smooth and engaging
 animations that enhance the user experienc
